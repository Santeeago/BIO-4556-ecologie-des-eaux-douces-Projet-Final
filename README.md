# BIO-4556-ecologie-des-eaux-douces---Projet-Final
Repository pour le devoir final du cours BIO 4556.

Lakes.r contient le code utilisé pour traiter les données recueillies pendant la deuxième sortie du terrain à Saint-Hyppolite.

L'outil nécessite les deux fichiers .csv contenant les données dans le même dossier, et l'adresse des fichiers doit être spécifiée à l'aide de la fonction setwd().
